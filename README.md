# Fabio & Mariana Wedding

Site premium de casamento criado como um projeto novo e independente.

Este projeto nao depende, nao altera e nao reutiliza configuracoes de Nexuus/Nexuread.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy

O projeto esta preparado para Vercel com `vercel.json`.
Veja `DEPLOY.md`.
